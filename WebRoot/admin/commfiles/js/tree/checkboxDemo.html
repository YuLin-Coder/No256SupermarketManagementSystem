CheckBox  演示
  • CheckBox 父子关系配置
      ——修改以下配置，点击 Tree 的 CheckBox，看看变化
          CheckBox被选中时:  影响父;   影响子
      CheckBox被取消选中时:  影响父;   影响子
    •  setting.checkable = true;
    •  setting.checkType = ;
    •  
    •  被选择的节点共 0 个
    •  当前被修改选择状态的节点共 0 个
    •  只计算每次check后修改状态的节点： Yes;  No
  • CheckBox 状态说明
      当前节点没有被选择； 鼠标移到该节点上显示为：
         如果是父节点，则没有被选择的子节点。
      当前节点没有被选择； 鼠标移到该节点上显示为：
         存在被选择的子节点（只有父节点存在此状态）。
      当前节点被选择； 鼠标移到该节点上显示为：
         如果是父节点，则全部子节点都被选择。
      当前节点被选择； 鼠标移到该节点上显示为：
         存在没有被选择的子节点（只有父节点存在此状态）。
